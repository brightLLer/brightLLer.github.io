    (Leetcode)动态规划方程汇总

    Leetcode 算法 Leetcode 动态规划
    创建时间:2019-12-24 10:27
    字数:1.3k 阅读:
    评论:
    1. 不同路径：一个机器人位于一个 m x n 网格的左上角机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角。问总共有多少条不同的路径？
    2. 最小路径和：给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。
    3. 最大子序和：给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

      用 $dp[i]$ 存储以 $nums[i]$ 结尾的子序列中最大的一个.
    4. 爬楼梯：假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？
    5. 子集加和问题: 给定一个非负整数集，以及一个变量sum，判断是否存在一个给定集合的子集，使得该子集中的数的和等于n？
    6. 0-1背包问题：给出n个项目的weights和values，把这些项目装进一个容量为W的背包里，应该怎么装才能使得背包获得最大的价值？(给出两个整型数组val[0…n-1]和wt[0…n-1]用来分别代表n个项目的values和weights，同时，给出一个整数W来代表背包的容量，找出val[]的最大价值子集，使得该子集的权重小于或者等于W。)
    7. 找零钱：给定一张面额为N的钞票，我们现在想将其更换为同等价值的硬币，如果现在无限提供价值为S = {S1, S2, …, Sm}的硬币，那么我们可以用多少种方式来找零？（不关心硬币的找零顺序）
    8. 零钱兑换：给定不同面额(k种)的硬币 coins 和一个总金额 amount。编写一个函数来计算可以凑成总金额所需的最少的硬币个数。如果没有任何一种硬币组合能组成总金额，返回 -1。
    9. 编辑距离：给定两个字符串str1和str2还有如下几个可以在str1进行的操作。试找出可以将str1转换为str2的最小编辑（操作）次数。(a. insert; b. Remove; c. Replace)
    10. 最长公共子序列：给定两个序列，找出它们的最长公共子序列的长度。一个子序列进要求在顺序上一致，但不要求一定要连续。
    11. 最长递增子序列：找一个给定序列的最长子序列，子序列中的所有元素已经按照升序排列。

      用 $dp[i]$ 表示以 $i$ 结尾的最长递增子序列.
    12. 打家劫舍：你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。给定一个代表每个房屋存放金额的非负整数数组，计算你在不触动警报装置的情况下，能够偷窃到的最高金额。
    13. 判断子序列：给定字符串 s 和 t ，判断 s 是否为 t 的子序列。你可以认为 s 和 t 中仅包含英文小写字母。字符串 t 可能会很长（长度 ~= 500,000），而 s 是个短字符串（长度 <=100）。
    14. 数组的每个索引做为一个阶梯，第 i个阶梯对应着一个非负数的体力花费值 cost[i](索引从0开始)。每当你爬上一个阶梯你都要花费对应的体力花费值，然后你可以选择继续爬一个阶梯或者爬两个阶梯。您需要找到达到楼层顶部的最低花费。在开始时，你可以选择从索引为 0 或 1 的元素作为初始阶梯。(0-n-1为台阶索引，n为楼梯顶部)

    15. 给定一个整数数组 nums ，找出一个序列中乘积最大的连续子序列（该序列至少包含一个数）。

      $Ma[i]$ 表示以 $nums[i]$ 结尾时的乘积最大值, $Mi[i]$ 表示以 $nums[i]$ 结尾时的乘积最小值.

    16. 买卖股票问题（注意买一次卖一次算一次交易, $k=1, n, +\infty$, 手续费, 冷冻期多种题型通用解法）：https://leetcode-cn.com/problems/best-time-to-buy-and-sell-stock/solution/yi-ge-fang-fa-tuan-mie-6-dao-gu-piao-wen-ti-by-l-3/

    17. 最长公共子串, 用dp[i][j]表示以X[0:i]和Y[0:j]的最长公共子序列.
