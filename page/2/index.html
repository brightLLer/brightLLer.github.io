    关于本站的一点小说明

    为什么开始写博客

    之前学习的时候一直有做笔记的习惯，但一直不成体系，各种平台纸质材料都尝试去用过，但感觉自己就是坚持不下来，这次想着借助搭建这么一个小博客平台的机会，把自己学过的东西，
    或者自己日常生活的点点滴滴给记录下来，大概就是这些吧。（想给首页凑一下字数都凑不出来，╮(╯▽╰)╭）。

    关于本站名称的由来

    Zeroone，好吧计算机专业的同学都知道01的意义，不过取这个名字的另外一个原因是假面骑士01开播，也算是个纪念吧。（没错我是假面粉╮(╯▽╰)╭）。

    博客平台搭建

    本博客通过 Hexo 生成，并部署在 GitHub Pages，主题[3-hexo]来源自github开源项目(https://github.com/yelog/hexo-theme-3-hexo)
    ，感谢开源项目的各位作者技术的支持，小弟渣渣不胜感激！！！

    • 搜索支持文章标题、标签(#标签)、作者(@作者)、全文搜索(in:内容)
    • pad/手机等移动端适配
    • 页面全局快捷键 3-hexo快捷键说明
